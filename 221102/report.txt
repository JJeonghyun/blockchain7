컴파일러 언어 : 언어 자체가 우리를 위한 것.
    - 즉, 컴퓨터는 언어를 읽지 못한다.
    - 우리의 언어를 컴퓨터가 읽을 수 있게 해주는 것이 컴파일러.
    - 기계어로 변환. Buffer < 전형적인 기계어
    - Buffer : 메모리 상에서 저장하는 형식 / 포멧
        - 데이터를 한 곳에서 다른 한 곳으로 전송하는 동안 일시적으로 그 데이터를 보관하는 메모리의 영역
        - raw binary 데이터를 저장할 수 있는 특수 객체
    - Build < 컴파일을 하여 프로그램을 생성한다
인터프리터 언어 : 양쪽 다 읽을 수 있는 언어
    - 컴파일러 언어와 차이점 : 컴파일러 언어는 항상 컴파일을 실행해야 확인 가능, 인터프리터 언어는 그냥 바로 실행 할 수 있다
