background-color: 배경색 넣기

background-color: url(#) => 배경 이미지의 크기 지정, 이미지가 다 보이도록 하되 최대 크기 (비율 맞춤)

background-size: contain => 배경 이미지의 크기 지정, 이미지가 다 보이도록 하되 최대 크기 (비율 맞춤
background-size: cover => 배경 이미지 크기 지정, 최대 크기(비율 맞춤), 이미지가 짤릴 수 있음

background-size: 100px 100px; => 배경 이미지 사이즈, 너비 높이 순

background-position: center => 배경 이미지 위치 가운데 정렬

background-repeat:  = >배경 이미지 반복 여부, 반복한다. 기본 설정값 이다.
background-repeat: no-repeat; =>배경 이미지 반복 여부, 반복하지 않는다
background-repeat: repeat-x; =>배경 이미지 반복 여부, x축으로만 반복 하겠다.
background-repeat: repeat-y; =>배경 이미지 반복 여부, y축으로만 반복 하겠다.


border-radius : 10px => 외곽선의 모서리를 깎는다. 반지름을 입력한다
border: 1px solid black; => 외곽선을 그린다. 정말 자주 쓰는 것이다
border: 10px dashed red; => 외곽선을 점선으로 그린다
border: 10px double blue; => 외곽선을 두줄로 그린다
border: 10px dotted violet; => 외곽선을 점으로 찍는다
border: none; => 외곽선 삭제



display: flex;
ustify-content: center; => flex의 주축이라고는 하는데...
                           진행하는 방향에 대한 정렬 방식이다.

justify-content: flex-start; => flex가 시작하는 위치(왼쪽)
justify-content: flex-end; => flex가 끝나는 위치(오른쪽)
justify-content: space-between; => 양끝을 맞추고 나머지는 균등 정렬
justify-content: space-around; => 균등 정렬인데, 양끝에 여백이 적다.
justify-content: space-evenly; => 그냥 균등 정렬(겉으로 보기에 모든 margin이 같다) 

align-items: center; => flex의 진행하는 방향의 직각, 수직에 대한 정렬
align-items: stretch; => 늘려준다.

flex-wrap: wrap; => 넘쳤을 때 줄이지 않고 다음줄로 
                      기본 값은 nowrap

flex-direction: row; => 가로 진행 방향
flex-direction: column; => 세로 진행 방향
